<h2>About the Project</h2>

<p>
SpendReality is a frontend web application designed to help users understand the true cost of their purchases by translating money into <b>time and effort</b>.
Instead of just showing price, it calculates how many hours or days of work are required to afford an item based on user income.
</p>

<p>
This project was built to explore how financial decisions feel when represented in a more human and intuitive way.
</p>

<h2>Features</h2>

<ul>
  <li>Converts product price into working hours required</li>
  <li>Supports both hourly wage and monthly salary input</li>
  <li>Automatic income normalization system</li>
  <li>Monthly financial impact analysis</li>
  <li>Low / Medium / High spending impact classification</li>
  <li>Dark mode UI support</li>
  <li>Interactive modal-based report system</li>
</ul>

---

<h2>How It Works</h2>

<pre>
User Input тЖТ Income Normalization тЖТ Calculation Engine тЖТ Report Generation тЖТ UI Display
</pre>

<ul>
  <li>Inputs price and income details</li>
  <li>Converts salary into comparable hourly format</li>
  <li>Calculates work hours required for purchase</li>
  <li>Displays financial impact visually in a modal</li>
</ul>
